## Installation & Running API

```bash
# install dependencies
$ npm install

# running api
$ npm run start:dev
```

## Testing

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
# note: open index.html file located in: /coverage/lcov-report/index.html
```
